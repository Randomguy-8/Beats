# WIP

### Beats
## A bash based music player!





### Depedendencies

- mpg321

### Usage
``` 
-a -- About
-h -- Help
-b -- Play Music directly 
```
<br>
## Installation
<b>Through Terminal</b>
1) Open terminal!
2) `cd` to the tool.
3) Type `chmod +x beats` to give it all executable permissions.
4) Now type `./beats` to run.<br>
