### Beats !||!
A konsole based music player!
<br>

### Depedendencies

- mpg321

### Usage
``` 
-a -- About
-h -- Help
-b -- Play it directly 
```

### How to Install?
<b>By Terminal</b>
1) Open terminal!
2) `cd` to the tool.
3) Type `chmod +x beats` to give it all executable permissions.
4) Now type `./beats` to install .<br>
